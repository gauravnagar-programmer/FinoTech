FinoTech is a simple Hindi-first financial awareness and planning web app designed

Along with personal finance guidance, FinoTech introduces a powerful community-based

reduces dependency on local moneylenders, and builds a transparent, trust-based

This prototype was built overnight for a hackathon to demonstrate how rural finance 
can be made simple, community-driven, safe, and accessible for everyone.
